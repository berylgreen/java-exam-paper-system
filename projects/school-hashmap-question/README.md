# 映射字典 (HashMap) (教务系统)

教务系统中，学生信息原本保存在 `List<Student>` 中。由于学号只是 `Student` 对象的一个属性，当需要根据学号查询或删除学生信息时，通常要遍历整个 `List` 逐个比较学号，效率较低。

请使用 `HashMap` 对该设计进行优化，要求如下：  
(1) 使用 `HashMap<String, Student>` 存储学生数据，其中键为学生学号，值为对应的 `Student` 对象；  
(2) 实现以下功能：
   - 添加学生信息；
   - 根据学号查询学生信息；
   - 根据学号删除学生信息；  
(3) 编写 `Main` 测试类（或在 `main` 方法中测试），演示上述功能的使用效果，并体现 `HashMap` 可以通过键快速查找数据的特点。
**测试数据示例：**
- 添加课程：
  - `101`，`高等数学`
  - `102`，`大学物理`
  - `103`，`计算机科学`
- 查询课程：`102`
- 删除课程：`102`
- 删除后再次查询：`102`

**预期输出示例：**
```text
添加后课程数量：3
查询 id=102 的课程：大学物理
删除后再次查询 id=102：未找到
```
