# 映射字典 (HashMap) (物流系统)

物流系统原先使用一个 `List` 保存包裹信息，在根据包裹 ID 查找包裹时，需要顺序遍历列表，效率较低。

请使用 `HashMap` 对该功能进行优化，完成以下要求：  

(1) 使用 `HashMap<String, Package>` 存储数据，其中键为包裹 ID，值为对应的 `Package` 对象。  
(2) 实现以下基本操作：
   - 存入一个包裹信息
   - 根据 ID 查询包裹信息
   - 根据 ID 删除包裹信息  
(3) 编写 `Main` 测试类（或在 `main` 方法中测试），验证可以通过 ID 快速完成查找、插入和删除操作。

要求程序结构清晰，能够体现 `HashMap` 相比顺序遍历在按键查找场景中的优势。

## 测试数据示例
请在 `Main` 类中使用以下测试数据进行演示：
- 添加包裹：
  - `101`，`电子产品包裹`
  - `102`，`书籍包裹`
  - `103`，`衣物包裹`
- 查询包裹：`102`
- 删除包裹：`102`
- 删除后再次查询：`102`

**预期输出示例：**
```text
添加后包裹数量：3
查询 id=102 的包裹：书籍包裹
删除后再次查询 id=102：未找到
```
