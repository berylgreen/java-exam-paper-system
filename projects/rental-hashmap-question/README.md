# 映射字典 (HashMap) (租车系统)

租车系统原先使用一个 `List<Vehicle>` 保存车辆信息，在根据 ID 查询或删除车辆时，需要遍历列表查找，数据量较大时效率较低。

请使用 `HashMap` 对该功能进行优化，要求如下：
1. 使用 `HashMap<String, Vehicle>` 存储车辆信息，其中 Key 为车辆 ID，Value 为对应的 `Vehicle` 对象。
2. 实现以下功能：
   - 添加车辆信息
   - 根据 ID 查询车辆信息
   - 根据 ID 删除车辆信息
3. 编写 `Main` 测试程序，演示上述功能的使用，并体现 `HashMap` 在按键查找时的高效性。
**测试数据示例：**
请在 `Main` 类中使用以下测试数据进行演示：
- 添加车辆：
  - `101`，`丰田卡罗拉`
  - `102`，`本田雅阁`
  - `103`，`宝马X5`
- 查询车辆：`102`
- 删除车辆：`102`
- 删除后再次查询：`102`

**预期输出示例：**
```text
添加后车辆数量：3
查询 id=102 的车辆：本田雅阁
删除后再次查询 id=102：null
```
