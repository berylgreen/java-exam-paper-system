# 试卷恢复工具 (Recover Exam Paper)

此目录下包含用于从导出的 Word 试卷（`.docx`）文件中恢复题库对应试卷的脚本工具。

## 使用场景

如果您曾经导出了包含题目的 Word 试卷，但在系统中该试卷被误删，可以使用此脚本从 Word 文档中提取题目内容，并与当前数据库中的题库进行模糊匹配，最终将匹配成功的题目重新组装为新试卷保存回系统中。

## 依赖
- Python 3
- 启动并运行的后台系统 (默认访问 `http://localhost:18080`)

## 使用方法

将需要恢复的 Word 试卷文件（如 `试卷_44.docx`）放置于任意目录或当前目录中，然后执行：

```bash
python3 recover_exam_paper.py "试卷_44.docx"
```

脚本将自动：
1. 解析 `docx` 文件，提取文本和题干。
2. 从本地后台拉取所有可用题目。
3. 对题干进行清洗和相似度计算匹配。
4. 匹配成功后，自动在系统中生成一份名为原文件标题（或“恢复的试卷”）的新试卷。

请注意：若由于题库发生过重大变更导致匹配度过低，部分题目可能会跳过，请留意控制台的输出结果。
