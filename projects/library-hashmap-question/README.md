# 映射字典 (HashMap) (图书馆系统)

图书馆系统原先使用一个 `List<Book>` 保存图书信息。根据图书编号查询时，需要顺序遍历列表，当数据量较大时查询效率较低。

请使用 `HashMap` 对该功能进行优化，要求如下：
1. 使用 `HashMap<String, Book>` 存储图书信息，其中键为图书编号（ID），值为对应的 `Book` 对象。
2. 实现以下功能：
   - 添加图书信息
   - 根据 ID 查询图书
   - 根据 ID 删除图书
3. 编写 `Main` 类进行测试，演示 `HashMap` 在按 ID 查找时的便捷性。

请给出完整的 Java 实现代码。
**测试数据示例：**
请在 `Main` 类中使用以下测试数据进行演示：
- 添加图书：
  - `101`，`Java编程思想`
  - `102`，`算法导论`
  - `103`，`计算机网络`
- 查询图书：`102`
- 删除图书：`102`
- 删除后再次查询：`102`

**预期输出示例：**
```text
添加后图书数量：3
查询 id=102 的图书：算法导论
删除后再次查询 id=102：null
```
