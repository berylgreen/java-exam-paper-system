# 映射字典 (HashMap) (酒店系统)

酒店管理系统原先使用 `List<Room>` 保存房间数据。由于 `List` 在根据房间编号（ID）查询或删除房间时，通常需要从头到尾遍历并逐个比较编号，随着房间数量增加，查询和删除效率会降低。

请使用 `HashMap` 对该设计进行优化，要求如下：  

(1) 使用 `HashMap<String, Room>` 保存酒店房间数据，其中键为房间编号（ID），值为对应的 `Room` 对象。  
(2) 实现以下功能：
   - 添加房间数据；
   - 根据房间编号查询房间信息；
   - 根据房间编号删除房间信息。  
(3) 编写 `Main` 测试类，演示上述功能的使用过程，并体现使用 `HashMap` 后可以更方便、高效地按编号查找房间。

## 测试数据示例
请在 `Main` 类中使用以下测试数据进行演示：
- 添加客房：
  - `101`，`总统套房`
  - `102`，`豪华大床房`
  - `103`，`标准间`
- 查询客房：`102`
- 删除客房：`102`
- 删除后再次查询：`102`

**预期输出示例：**
```text
添加后客房数量：3
查询 id=102 的客房：豪华大床房
删除后再次查询 id=102：未找到
```
