# 映射字典 (HashMap) (电商系统)

电商系统原先使用两个相互对应的 `List` 分别保存订单编号和 `Order` 对象。由于根据订单编号查询时需要遍历整个列表，当数据量增大时性能较差。

请使用 `HashMap` 对该存储方式进行重构，其中：
- Key 为订单编号（`String` 类型）
- Value 为订单对象（`Order` 类型）

要求完成以下功能：
1. 实现订单的添加功能；
2. 实现根据订单编号查询订单功能；
3. 实现根据订单编号删除订单功能；
4. 编写 `Main` 测试类，演示 `HashMap` 的基本使用，并说明其在按键查找时相较顺序遍历更高效。
