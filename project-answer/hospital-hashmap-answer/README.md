# 映射字典 (HashMap) (医疗系统)

医疗系统原先使用 `List` 保存患者信息，每次根据患者编号查询时都需要遍历查找，数据量较大时效率较低。

请使用 `HashMap` 对该存储方式进行优化，要求如下：

1. 使用 `HashMap<String, Patient>` 存储患者数据，其中键为患者编号，值为对应的 `Patient` 对象。
2. 实现以下功能：
   - 添加患者信息
   - 根据患者编号查询患者信息
   - 根据患者编号删除患者信息
3. 编写 `Main` 测试类（或在 `main` 方法中编写测试代码），演示上述功能，并体现 `HashMap` 按键查找更高效、更直接的特点。
